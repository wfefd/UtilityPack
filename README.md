## 1. 과제 소개: UtilityPack

`UtilityPack`은 팀원이 각각 구현한 5가지 유틸리티 기능을 하나의 파이썬 프로젝트 안에서 모듈 형태로 모아 둔 프로그램입니다.  
각 기능은 **자기 파일에 `main()` 함수를 가지는 터미널 기반 프로그램**이며,  
메인 메뉴(`main_menu.py`)에서 해당 모듈을 호출하는 방식으로 연동할 수 있도록 설계되었습니다.

### 기능 구성

- **단위 변환기 (`unit_converter_hyungkyu.py`)**  
  길이, 면적, 부피, 속도, 시간에 대해 자주 사용하는 단위들만 추려 간단하게 변환할 수 있는 도구입니다.  
  카테고리와 단위를 번호로 선택하고, 값을 입력하면  
  `입력값 from 단위 = 결과값 to 단위` 형식으로 변환 결과를 터미널에 출력합니다.

- **간단 계산기 (`simple_calculator_jincheol.py`)**  
  두 숫자와 연산자(+, -, *, /)를 입력받아 사칙연산을 수행하는 콘솔 계산기입니다.  
  잘못된 입력이나 0으로 나누기 같은 상황에 대해 예외 처리를 하고, 계산 결과를 텍스트로 출력하는 형태로 설계되어 있습니다.

- **점수 평균 계산 (`score_average_harin.py`)**  
  `80 90 75.5 100` 처럼 공백으로 구분된 여러 점수를 한 줄로 입력받아 평균을 계산합니다.  
  내부에서 실수 리스트로 변환한 뒤 합계와 개수를 사용해 평균을 구하고,  
  점수가 하나도 없을 경우에는 `"입력 오류"` 메시지를 출력하도록 구현되어 있습니다.

- **문자열 처리 (`string_tools_seunghwan.py`)**  
  문자열을 대문자/소문자로 변환하거나, 뒤집거나, 모음 개수를 세거나, 공백을 제거하는 등의 간단한 텍스트 유틸 기능을 모아놓은 모듈입니다.  
  별도의 메뉴 코드에서 이 모듈을 불러, 사용자가 문자열과 기능(대문자로 변환, 소문자로 변환, 뒤집기, 모음 개수 세기, 공백 제거 등)을 선택해 실행하는 방식으로 사용할 수 있습니다.

- **랜덤 비밀번호 생성기 (`password_generator_jungsu.py`)**  
  원하는 길이를 숫자로 입력하면 그 길이에 맞는 랜덤 비밀번호를 생성하는 콘솔 프로그램입니다.  
  소문자, 대문자, 숫자, 특수문자(`!@#$%^&*`)를 조합해 문자열을 만들고,  
  `<길이>자리 비밀번호: <생성된 비밀번호>` 형식으로 터미널에 결과를 출력합니다.  
  잘못된 입력(숫자가 아닌 값, 0 이하의 숫자 등)에 대한 간단한 예외 처리도 포함되어 있습니다.

## 3. 실행 방법

1. 저장소 클론

   ```bash
   git clone <레포지토리 URL>
   cd UtilityPack
   ```

2. 파이썬 실행(Python 3.x)
   
  ```bash
  python main_menu.py
  ```

### 4-3. Merge & Conflict 해결 경험

- 공용 파일에서 여러 차례 충돌(Conflict) 발생 경험
- 주요 충돌 파일: `main_menu.py`
  - 여러 기능을 한 메뉴에서 호출하는 과정에서, 서로 다른 브랜치에서 같은 부분(메뉴/버튼 영역)을 수정하여 충돌 발생
- 해결 절차:
  1. `git pull origin main`으로 최신 `main` 내용 반영
  2. 충돌 표시 구간(`<<<<<<<`, `=======`, `>>>>>>>`) 확인
  3. 각 기능의 버튼 및 메뉴 항목을 모두 포함하도록 수동 병합
  4. 로컬에서 프로그램 실행 후 전체 메뉴 클릭 테스트
  5. 정상 동작 확인 후 커밋 및 PR 업데이트
- 프로젝트 소개/README 관련 텍스트에서도 일부 충돌 발생
  - 팀원별로 추가한 설명을 합쳐서 자연스럽게 이어지도록 수동 수정
